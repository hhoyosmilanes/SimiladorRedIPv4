Instrucciones de Red:

Debe configurar tres enlaces con la cantidad de hosts posibles especificada:

  • Enlace 1 - proveedor a ISP (IP Pública): N hosts posibles.
  • Enlace 2 - ISP a Hogar (IP Privada): N hosts posibles.
  • Enlace 3 - Hogar a Compu. (IP Privada): N hosts posibles.

Calcule la máscara de red y el prefijo (CIDR) adecuados para cada enlace según la cantidad de hosts.
